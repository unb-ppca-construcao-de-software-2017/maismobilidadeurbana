/*========= Description =========*/

Stride lite is a clean, modern and fully responsive WordPress Personal Blog theme built using Bootstrap 3 which makes it optimized for tablets and mobile.x

/*========= Installation =========*/

You can install the theme through the WordPress installer under "Themes" > "Install themes" by searching for "stride".

Alternatively you can download the file, unzip it and move the unzipped contents to the "wp-content/themes" folder of your WordPress installation. You will then be able to activate the theme.

/*========= Theme Features =========*/

* Bootstrap 3 integration
* Responsive design
* Internationalized & localization
* Drop-down Menu(limited to level one).
* Cross-browser compatibility
* Threaded Comments
* Gravatar ready
* Font Awesome
